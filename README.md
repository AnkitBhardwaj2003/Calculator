# 🚀 Advanced Animated Calculator

A modern, responsive, and fully animated calculator web application built using HTML, CSS, and JavaScript.

This project combines a clean glassmorphism UI with smooth animations, scientific calculator functionality, keyboard support, sound effects, dark/light mode switching, and a calculation history panel to deliver an interactive user experience.

## ✨ Features

### 🧮 Core Calculator Features
- Basic arithmetic operations
- Percentage calculations
- Decimal support
- Delete and clear functionality

### 🔬 Scientific Calculator
- `sin()`
- `cos()`
- `tan()`
- `sqrt()`
- `log()`
- Power operations

### 🎨 User Experience
- Smooth hover and click animations
- Floating calculator animation
- Neon glow effects
- Glassmorphism interface
- Fully responsive layout

### ⚡ Additional Functionalities
- Dark / Light mode toggle
- Keyboard support
- Sound effects on button clicks
- Calculation history panel
- Clear history option
